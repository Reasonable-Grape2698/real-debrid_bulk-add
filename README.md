# Instructions
Run add.sh 
```
./add.sh -a APIKEY -dir /torrents/dir
```
flags:
  -a API key (REQUIRED)
  -dir (Directory with .torrents, default /torrents, don't include ending slash)

Run select.sh
```
select.sh -f /torrents/dir/torrentIds.txt
```
flags:
  -f (torrentIds.txt location, default /torrents/torrentIds.txt)
